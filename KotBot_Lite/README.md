# KotBot Lite

Простой бот для подготовки контента под VK: видео, фото, плашки, watermark, фильтры, музыка, паблики и отдельные папки готовых файлов.

## Запуск

1. Распакуй архив.
2. Запусти `run.bat`.
3. Если нужны ошибки в консоли - запусти `run_debug.bat`.
4. Для ярлыка на рабочем столе запусти `Создать ярлык на рабочем столе.bat`.

## Главное

Видео всегда приводится к вертикальному формату `1080x1920` без черных полос. Если исходник другой формы, он заполняет кадр и обрезается по центру. Масштаб видео можно менять во вкладке **Видео**, а разброс масштаба - во вкладке **Рандом**.

## Паблики VK

Во вкладке **Паблики** можно выбрать паблик, например:

- Хвостик
- Клевер
- Паблик 3

У каждого паблика свои настройки:

- watermark;
- фильтры;
- плашка;
- текст;
- музыка;
- рандомизация;
- быстрый предпросмотр и безопасные зоны VK.

У каждого паблика теперь есть своя папка готовых файлов:

- `data/publics/hvostik/готовые_видео`
- `data/publics/hvostik/готовые_фото`
- `data/publics/klever/готовые_видео`
- `data/publics/klever/готовые_фото`

Когда выбран паблик **Хвостик**, готовые видео и фото сохраняются именно в его папку. Когда выбран **Клевер** - в папку Клевера.

## Автоимя файлов

Готовые файлы называются автоматически:

- `hvostik_20260626_001.mp4`
- `hvostik_20260626_002.mp4`
- `klever_20260626_001.jpg`

Номер сам увеличивается внутри папки выбранного паблика.

## Видео

Видео умеет:

- брать файл кнопкой;
- брать первый файл из `data/source_videos`;
- переключаться на предыдущее и следующее видео;
- отправлять неудачный ролик в `data/trash_videos` кнопкой 🗑;
- показывать предпросмотр;
- перематывать предпросмотр ползунком;
- обрабатывать одно видео;
- обрабатывать все видео из папки пачкой;
- переносить исходники из `data/source_videos` в `data/used_videos` после обработки.

## Безопасные зоны VK

Во вкладке **Видео** есть галочка **Показать безопасные зоны VK**. Это только рамки в предпросмотре. В готовое видео они не попадают.

## AI подписи

Во вкладке **AI подписи** можно сделать варианты подписей без Ollama.

Что есть:

- выбор AI провайдера;
- **Gemini Free** - основной бесплатный вариант через Google AI Studio;
- **OpenRouter Free** - бесплатный роутер моделей OpenRouter;
- **DeepSeek** - отдельный провайдер DeepSeek;
- **OpenAI** - старый вариант, если доступен;
- **Локальный шаблон** - без API ключа и без интернета, простые заготовки;
- поле для API ключа;
- поле модели;
- стиль подписи;
- описание сцены или идея;
- генерация 3-20 вариантов, по умолчанию 10;
- варианты теперь показываются крупными карточками, а не узким списком;
- один клик выбирает вариант;
- двойной клик по карточке или кнопка **Вставить** сразу переносит текст в подпись видео;
- кнопка **Копировать** копирует выбранный вариант;
- лишние строки-рассуждения от бесплатных моделей автоматически вычищаются.

Ключ сохраняется только локально в `config.json` на твоем компьютере. В архиве ключа нет.

## Фото

Фото находится в отдельной большой вкладке. Подпись-окошко на странице фото убрано. Фото работает отдельно от видео и сохраняет результат в папку выбранного паблика.

## Музыка

Треки можно складывать в `data/music`. Во вкладке **Музыка** есть выпадающий список треков, случайный трек, громкость оригинала и громкость музыки.


## Отдельная вкладка Папки

В верхнем меню появилась отдельная большая вкладка **Папки**. Там вынесено самое частое:

- исходники видео и фото;
- готовые видео и фото выбранного паблика;
- использованные видео и фото;
- хлам видео;
- папки ресурсов: watermark, PNG плашки, шрифты, музыка;
- папка всех VK-пабликов;
- быстрые действия: следующее видео, обработать все видео, обновить музыку.

Старые кнопки папок внутри настроек остались, но теперь для папок есть отдельное нормальное место.

## Папки

Основные папки:

- `data/source_videos` - исходники видео;
- `data/source_photos` - исходники фото;
- `data/used_videos` - использованные видео;
- `data/used_photos` - использованные фото;
- `data/trash_videos` - хлам видео;
- `data/watermarks` - watermark PNG;
- `data/plaques` - PNG плашки;
- `data/fonts` - шрифты;
- `data/music` - музыка;
- `data/publics` - папки пабликов VK.

## Настройки

Все настройки сохраняются в `config.json` рядом с `main.py`.

## Обновление 1.18

- Полностью переделана **Толщина букв**.
- Теперь жирность рисуется через отдельную маску текста, поэтому работает на любых шрифтах, включая кастомные TTF/OTF.
- Значение `1` уже дает заметное утолщение.
- Текст больше не уменьшается из-за жирности так сильно, поэтому эффект не пропадает.
- Толстый текст ограничивается областью плашки и не вылезает за PNG плашку.

## Обновление 1.19

Исправлена главная проблема рендера текста: предпросмотр и готовое видео теперь используют один и тот же финальный рендер.

Что изменилось:

- предпросмотр больше не масштабирует отдельно настройки текста;
- сначала рисуется настоящий финальный кадр, потом он уменьшается для окна;
- толщина букв, перенос строк, межбуквенное расстояние и позиция плашки должны совпадать с экспортом;
- толщина букв стала аккуратнее: ползунок 0-12 больше не раздувает буквы в огромные пятна;
- автоподбор размера теперь учитывает реальную толщину букв, чтобы текст не распирало после экспорта.

Если предпросмотр стал чуть тяжелее, это нормально: зато он честно показывает то, что попадет в готовое видео.
