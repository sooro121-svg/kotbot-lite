# Roadmap KotBot Lite

## Сделано

- Видео 1080x1920 без черных полос.
- Масштаб видео и рандом масштаба.
- Текст, шрифты, толщина букв, цвета кодом и палитрой.
- PNG плашка, позиция текста внутри плашки.
- Watermark.
- Фильтры видео и фото.
- Музыка из выпадающего списка.
- Громкость оригинала и музыки.
- Фото в отдельной вкладке.
- Паблики VK с отдельными настройками.
- Отдельные папки готовых файлов для каждого паблика.
- Автоимя файлов.
- Предыдущее/следующее видео.
- Хлам видео.
- Пакетная обработка всех видео из папки.
- Безопасные зоны VK в предпросмотре.
- Отдельная большая вкладка Папки.
- AI подписи без Ollama: Gemini Free, OpenRouter Free, DeepSeek, OpenAI и локальные шаблоны.
- Быстрые кнопки для исходников, готовых, использованных, хлама и ресурсов.
- Скрытый запуск без консоли и ярлык.

## Дальше можно добавить

- История обработок в отдельной таблице.
- Кнопка сделать 3 варианта одного ролика.
- Отдельные пресеты текста внутри паблика.
- Подготовка к публикации VK: токен, пост, загрузка видео/фото, история публикаций.
- Проверка перед экспортом: видео, watermark, музыка, папка готовых.

## Сделано в 1.18

## 1.19

- Сведен предпросмотр и экспорт к одному рендеру кадра.

- Утолщение букв ограничено до аккуратного диапазона, чтобы подписи не превращались в пятна.

## 1.20

- Добавлен ползунок Максимум строк для видео и фото.
- Добавлена кнопка Открыть выбранное видео в предпросмотре.
- Двойной клик по предпросмотру открывает выбранное видео в системе.
- Исправлен рандом масштаба при базе 100%: теперь он не залипает на 100.

## 1.21

- Кнопка предпросмотра переименована в Показать в папке.
- Двойной клик по предпросмотру теперь открывает проводник с выделенным исходным видео.

- В ручном режиме длинные строки клипуются областью плашки, а не блокируют движение всего текста.
